### [预览地址](https://learningwhy.today)

## 1.克隆项目
```
git clone https://github.com/Martin-Mas/VueExperience.git
```

## 2.进入项目
```
cd vue-experience
```

## 3.安装依赖
```
npm install
```

## 4.启动服务
```
npm run serve
```

## 5.打包发布项目
```
npm run build
```
