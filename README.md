# vue-experience

## 克隆项目
```
git clone https://github.com/Martin-Mas/VueExperience.git
```

## 进入项目
```
cd vue-experience
```

## 安装依赖
```
npm install
```

### 启动服务
```
npm run serve
```

### 打包项目
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
