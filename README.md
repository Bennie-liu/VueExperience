# vue-experience
该项目个人打算所有功能模块自己都尽量去写一遍，并不会直接套用现成代码。造车轮子去咯！

### [预览地址](https://learningwhy.today)

## 1.克隆项目
```
git clone https://github.com/Martin-Mas/VueExperience.git
```

## 2.进入项目
```
cd vue-experience
```

## 3.安装依赖
```
npm install
```

## 4.启动服务
```
npm run serve
```

## 5.打包发布项目
```
npm run build
```
